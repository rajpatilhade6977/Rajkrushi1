
# Rajkrushi AI Ultra App

Advanced Smart Farming & Dairy Management Platform

Modules:
- OTP Login (Firebase Phone Authentication)
- Animal Management
- Milk Production Records
- Milk Graph Analytics (Chart.js)
- Customer Milk Billing
- Farmer Marketplace (OLX Style)
- Farmer Social Network
- Cow Heat Detection
- AI Cow Disease Detector (image upload placeholder)
- Weather Smart Alerts
- Vet Doctor Finder (Google Maps)
- Government Scheme Alerts (placeholder)
- Admin Analytics Dashboard

Setup:
1. Create Firebase project
2. Enable Authentication (Phone)
3. Enable Firestore Database
4. Put Firebase config inside firebase.js
5. Open login.html

Run locally using Acode / VS Code / GitHub Pages.
